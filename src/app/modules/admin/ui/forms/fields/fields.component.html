Form Fields

By default, Treo changes the default form field appearance to fill and heavily modifies it to provide a more unique and consistent look. We recommend NOT using any other form field styles as they are not optimized for Treo.

These examples also use TailwindCSS utility classes. We strongly recommend you to use TailwindCSS to build your form layouts. This not only allows an extreme flexibility on your forms but it also saves time since forms are generic components and they don't necessarily require explicit id/class naming when using them with Angular because of how Angular handles the forms.
